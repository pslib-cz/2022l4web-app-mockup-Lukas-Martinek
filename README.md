# Mobilní aplikace pro analýzu dat ze Spotify
Mobilní aplikace bude brát data ze spotify, konkrétně nejposlouchanější umělce, skladby a žánry v určených časových úsecích (1 mešíc, 3 měsíce, půl roku rok).
